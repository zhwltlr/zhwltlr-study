# Tailwind

## Getting Started

First, install the tailwind

```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init
```

And, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Description

#### 1. Tailwind

- page.tsx, layout.tsx를 활용
- [id] 폴더로 동적 라우팅 확인

#### 2. React-hokk-form

- React-hook-form을 이용하여 간단한 로그인 페이지 구현
- 유효성 검증 포함
